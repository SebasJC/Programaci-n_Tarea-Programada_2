Este es un programa en que se lee una lista de empleados de un archivo, se calcula su salario según el tipo que sea, se ordena en forma de árbol y se imprime un reporte en un archivo separado por compas.

Para compilar el programa, ejecute `make`.

Para correr el programa, ejecute `./bin/empleado`.

Para compilar los tests, ejecute `make test`.

Para correr los tests, ejecute `./bin/tests`.
